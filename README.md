# Smart Calculator App (Streamlit + Groq LLM)

This is a beginner-friendly calculator app built with **Python**, **Streamlit**, and **Groq LLM**.  
It runs locally (or in Google Colab + Cloudflare Tunnel) and can:

- Perform basic math operations
- Calculate power & percentage
- Explain steps using Groq LLM (llama-3.3-70b-versatile)
- Accept natural language math questions
- Maintain a visible calculation history
